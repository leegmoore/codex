# codex-ts

TypeScript port of codex-rs.

## Structure

This project mirrors the structure of codex-rs, with each Rust module ported to idiomatic TypeScript.

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Build
npm run build

# Type check
npm run type-check
```

## Modules

- `utils/string` - String utility functions for UTF-8 handling
